📘 Ledgora – Use Cases, Features & Functionalities
🧠 1. Core Purpose

Ledgora is an offline-first personal ledger app that helps users:

• Track income and expenses
• Organize transactions by categories
• Monitor balance and cash flow
• Generate professional reports
• Analyze spending patterns
• Set payment reminders
• Securely backup & restore data (premium)

All with a simple, fast, and modern experience.

👤 2. User Types
Free User

• Can use all basic ledger features
• Sees ads
• Limited advanced features

Premium User (One-Time Purchase)

• No ads
• Unlimited reports
• Advanced analytics
• Google Drive backup & restore
• More category creation
• Enhanced experience
